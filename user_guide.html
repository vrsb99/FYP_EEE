User Guide

Welcome to the Optimal Portfolio Allocation tool. This guide will help you navigate and effectively utilize the system to analyze and optimize asset allocations using various financial models.

Access the live demo of the application here. This link takes you to the main interface where you can interact with different components of the system.

Overview of Components

The system is divided into two main components: the backend, which handles data collection, preprocessing, and optimization algorithms, and the frontend, which provides the user interface for interaction.

Backend Components: Code for Generating Analysis Data

  • 00_download_all_files.py: Initiates the process by downloading all necessary datasets from DigitalOcean Spaces, ensuring that the frontend has access to the latest data for analysis.
  • 01_obtain_tickers.py: Downloads and stores ticker symbols from TopForeignStocks.com, which are crucial for retrieving financial data for various equities.
  • 02_obtain_data.py: Fetches and stores technical data from Yahoo Finance using the ticker symbols obtained, providing the raw data needed for subsequent analysis.
  • 03_ranking_equities.py: Ranks equities based on the Sharpe Ratio, a critical step in identifying top-performing stocks for portfolio optimization.
  • 04_test_models.py: Tests different allocation models for a quarterly period, allowing for the evaluation of various strategies under recent market conditions.
  • 05_validation.py: Conducts backtesting of each model on a rolling basis over a given period to validate their historical performance and robustness.
  • 06_upload_app.py: Uploads all generated data to DigitalOcean Spaces for backup and easy access across different platforms.
  • 07_review.ipnb: Notebook for reviewing the results of the models and generating graphs, facilitating a visual understanding of performance.
  • 08_other_backtest.ipynb: Used for reviewing additional backtesting results and generating detailed graphs for comprehensive analysis.

Frontend Components: Navigating the Website

  • Main Page: Provides a dashboard for real-time analysis of financial models and asset allocations.
  • Ranking Equities: Allows users to view and interact with the rankings of equities based on their Sharpe Ratios.
  • Studying Models: Enables detailed examination of various allocation models and their performance under different scenarios.
  • Model Validation: Offers tools to validate the performance of models over time using historical data.

Interacting with the Frontend

Use the sidebar to navigate between different pages. Each page provides interactive elements such as dropdowns, sliders, and buttons that allow you to customize the analysis according to your needs.

Viewing and Analyzing Results

Results of the analyses can be viewed directly on the frontend in the form of tables, charts, and graphs. These visualizations provide insightful perspectives on the data and model performances.
