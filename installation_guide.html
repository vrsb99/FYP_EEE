Installation Guide

This guide will assist you with the setup and installation of the necessary components for the Optimal Portfolio Allocation project.

Prerequisites

Before beginning the installation, ensure you have administrative access on your computer and a stable internet connection for downloading the necessary software and packages.

Step 1: Installing Python and Anaconda

You can choose to install Python directly or via Anaconda. Anaconda simplifies package management and deployment.

Option 1: Install Python Directly

  1. Visit the official Python download page at https://www.python.org/downloads/
  2. Download and install Python 3.11. Ensure to check 'Add Python 3.11 to PATH' during installation.

Option 2: Install Anaconda

  1. Download Anaconda for Python 3.11 from Anaconda's website.
  2. Follow the on-screen instructions to install Anaconda. Include Anaconda in your system's PATH to make it the default Python interpreter.

Step 2: Set Up the Project Environment

Create a virtual environment to manage the dependencies for the project separately from other Python projects.

Using Python

python -m venv myenv
myenv\Scripts\activate (Windows)
source myenv/bin/activate (Mac/Linux)

Using Anaconda

conda create --name myenv python=3.11
conda activate myenv

Step 3: Install Required Python Packages

Install all required packages specified in the requirements.txt file.

pip install -r requirements.txt

Step 4: Clone the Repository

Clone the project repository to your local machine using Git.

git clone https://github.com/vrsb99/FYP_EEE.git
cd FYP_EEE

Step 4: Environment Variables

Set up the necessary environment variables.

cp .env.example .env
# Edit .env file with your credentials

Step 6: Run the Application

Start the application using Streamlit.

cd frontend
streamlit run app.py

Follow these detailed steps to ensure a complete and functional setup of the Optimal Portfolio Allocation project on your system.
